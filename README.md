## Мета роботи

Розробити програмне рішення для автоматизованого розбору лог-файлів польотних контролерів Ardupilot, візуалізації маршруту та обчислення підсумкових метрик польоту. Інструмент має конвертувати сирі дані з датчиків (GPS, IMU) у зручний для аналізу формат та побудувати 3D-модель переміщення апарату.

## Як запустити

1. Скачати всі необхідні бібліотеки
   `pip install -r requirements.txt`
2. `cd main`
3. `streamlit run dashboard.py`

## Telemetry Parser — Модуль обробки даних

Цей модуль є "фундаментом" системи. Він відповідає за автоматизований розбір бінарних файлів телеметрії, очищення даних від шумів та підготовку структурованих масивів для візуалізації та аналізу.

### **Функціонал та логіка роботи**

* **Декодування MAVLink:** Пряме зчитування та парсинг бінарних повідомлень типів `GPS` та `IMU` за допомогою бібліотеки `pymavlink`.
* **Інтелектуальна фільтрація:**
    * Автоматичне відсіювання "сміттєвих" GPS-точок (валідація за кількістю супутників $\ge 6$ та точністю HDOP $< 5.0$).
    * Корекція нульових координат та перерахунок одиниць вимірювання у міжнародні стандарти (SI).
* **Аналіз метаданих:** Автоматичний розрахунок реальної частоти оновлення датчиків ($Hz$) та загальної тривалості активної фази польоту.

### **Технічні особливості**

* **Pandas Integration:** Усі дані конвертуються у `pandas.DataFrame`, що дозволяє миттєво проводити статистичний аналіз або будувати графіки.
* **Гнучкість:** Реалізована підтримка декількох інерціальних систем (IMU1, IMU2, IMU3) для аналізу стабільності роботи бортового обладнання.
* **Експорт:** Можливість швидкого збереження оброблених даних у формат `.csv`.

### **Використання через CLI**

Модуль можна використовувати як окремий інструмент для швидкого отримання даних без запуску Web-інтерфейсу:

```bash
# Базовий запуск для перевірки логу
python bin_parser.py path/to/flight.tlog

# Запуск із збереженням результатів у CSV
python bin_parser.py path/to/flight.tlog --save

## Drone Flight Analyzer — Web UI + AI Assistant

Веб-застосунок для аналізу польотних логів Ardupilot з AI-звітом про аномалії.

### Web UI

- Реалізовано на Streamlit
- Завантаження `.bin` файлу через `st.file_uploader`
- Автоматичне відображення метрик польоту у вигляді карток (тривалість, дистанція, швидкість, висота)
- Розгортання детальних метрик (вертикальна швидкість, прискорення, швидкість з IMU)
- Кнопка для генерації AI-звіту

### AI-асистент

- Використано Groq API + модель LLaMA 3
- На основі обчислених метрик формується промпт українською мовою
- AI аналізує дані та повертає текстовий висновок про аномалії, оцінку польоту та рекомендації
- API ключ зберігається в `.streamlit/secrets.toml` (безпечно, не вимагає введення від користувача)
